# キャラクタースプライト発注仕様書（4号機・Blender担当向け）

名鉄ボンタン狩り（incomebase0108-alt/meitetsu-bontan-game）のキャラを
CSS描画からBlenderレンダリングのPNGスプライトに差し替える。

**このフォルダに `<アーキタイプID>.png` を置くだけでゲームが自動で差し替える**（コード変更不要）。
無いファイルは現行CSSキャラのまま動くので、1体ずつ納品してOK。プロンプト案は同フォルダの PROMPTS.md も参考に。

## 共通仕様（必須）

| 項目 | 内容 |
|---|---|
| 形式 | PNG・背景透過 |
| サイズ | 高さ 800〜1000px、幅は実比（キャラ幅。目安 450〜550px）。ゲーム側で縮小表示 |
| 向き | **右向き**（横スクロール用。左向きはゲームが自動反転） |
| ポーズ | 直立〜軽い構えのidle 1枚。**足の裏が画像の最下端にぴったり接地**（余白を下に作らない） |
| ライティング | 均一・影は薄く（地面影はゲーム側で付ける） |
| スタイル | トゥーン/アニメ調レンダリング推奨。**ブロック調・マネキン調は禁止。顔のパーツ（目・眉・口）、髪の毛の造形、筋肉・服のしわまで作り込むこと**（過去の格闘ゲームでのhoshiさんフィードバック） |
| 世界観 | 全員「リーゼント＋改造学ラン（詰襟長ラン・金ボタン）＋ボンタン」の昭和ヤンキー。各キャラのモチーフで差別化 |

## 【追加発注】駅ボス23体の個別バリエーション（hoshiさん指示 2026-06-12）

上の16体とは別に、**駅ごとのボス専用スプライト**を作る。`assets/characters/boss/<駅id>.png` に置くと
その駅だけ専用画像になる（無い駅はアーキタイプ画像→CSSの順でフォールバック。優先度は駅専用が最上位）。

バリエーション軸（hoshiさん指定）：
- **特攻服姿**（刺繍入りロング特攻服。色は駅ごとに変える＝複数色）
- **体型差**：デブ／ヤセ／標準／巨漢を散らす
- **凶器持ち**：木刀・金属バット・チェーン・バール・熊手・巨大しゃもじ等（刃物・銃は不可、ゲームのノリに合わせて）

## 納品方法
1. 4号機がこのリポジトリに push できるなら `assets/characters/`（駅ボスは `assets/characters/boss/`）に直接コミット
2. できなければ PNG を zip にして Slack #claud に添付 → 1号機がコミットする

## 確認方法
リポジトリをクローンして `desktop-tutorial/index.html` をブラウザで開き、「新しく始める」→戦闘画面で差し替わっていればOK。
