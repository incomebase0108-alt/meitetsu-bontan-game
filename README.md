# 名鉄ボンタン狩り 〜蒲郡から新安城まで〜

名鉄蒲郡線+西尾線の全23駅を電車で巡り、各駅前でダブルドラゴン風のベルトスクロールアクション。雑魚不良を蹴散らし、駅ボスを倒してボンタン狩り（ズボン奪取）するブラウザゲーム。

公開版: https://incomebase0108-alt.github.io/meitetsu-bontan-game/

## 遊び方

1. `index.html` をブラウザで直接開く（ローカルファイル可）、または
2. 簡易サーバで配信：
   ```bash
   cd desktop-tutorial
   python3 -m http.server 8000
   # ブラウザで http://localhost:8000
   ```

### 操作
- **スマホ**: 左の仮想パッドで8方向移動＋右の3ボタン（👊パンチ/🦵キック/💥必殺）。移動しながら攻撃可（マルチタッチ）
- **PC**: 矢印キーで移動、**A**=パンチ / **S**=キック / **D**=必殺技

### ゲームの流れ
- 駅前ステージを右へ進む → 雑魚不良が2〜3人ずつ襲ってくる → 全滅させると先へ進める（GO ▶）
- ステージ最奥で**駅ボス**が登場（頭上にHPバー）。倒すとボンタン狩り→レベルアップ→次の駅へ
- **必殺ゲージ**: 攻撃ヒット・被弾で蓄積。MAXで💥が点灯し、周囲の敵をまとめて吹き飛ばす
- **キック・コンボ4発目・クリティカル**で敵をダウンさせられる。起き上がりは無敵
- ボスが光って力を溜めたら大技の予告 → 離れてかわせ！残りHP35%でボスはキレて凶暴化
- 蒲郡競艇場前ではレアエンカウント（極道）あり

## キャラクター
全員リーゼント＋改造学ラン（長ラン＋金ボタン＋ボンタン）のCSS描画スプライト。主人公は黒長ラン＋さらし、ラスボスは白ラン。雑魚は3種（チンピラ/取り巻き/鉄砲玉）×駅カラーの色違い。

## 路線・全23駅

### 名鉄蒲郡線 (蒲郡→吉良吉田) 10駅
蒲郡 → 蒲郡競艇場前 → 三河鹿島 → 形原 → 西浦 → こどもの国 → 東幡豆 → 西幡豆 → 三河鳥羽 → **吉良吉田（中ボス）**

### 名鉄西尾線 (吉良吉田→新安城) 13駅
吉良吉田 → 上横須賀 → 福地 → **西尾（中ボス）** → 西尾口 → 桜町前 → 米津 → 南桜井 → 桜井 → 堀内公園 → 碧海古井 → 南安城 → 北安城 → **新安城（ラスボス）**

## ファイル構成

```
desktop-tutorial/
├── index.html          メイン画面（仮想パッド・クレジット画面含む）
├── css/style.css       スタイル（不良スプライト・ベルトアクションUI）
├── js/
│   ├── stations.js     23駅 + 不良ボスのデータ
│   ├── scenes.js       駅前ステージ背景データ（CSSフォールバック用）
│   ├── sprites.js      リーゼント不良スプライト描画（共有）
│   ├── belt.js         ベルトスクロールアクションエンジン
│   ├── credits.js      写真クレジットデータ＋クレジット画面
│   ├── map.js          路線図UI
│   ├── audio.js        WebAudio合成のSFX＋BGM
│   ├── save.js         localStorageセーブ
│   └── game.js         状態管理・画面遷移
└── assets/bg/          駅写真（1400px/JPEG圧縮済み）
```
