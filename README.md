# 名鉄ボンタン狩り 〜蒲郡から新安城まで〜

名鉄蒲郡線+西尾線の全23駅を電車で巡り、各駅前で不良ボスと格闘、勝利後にボンタン狩り（ズボン奪取）するブラウザゲーム。

## 遊び方

1. `index.html` をブラウザで直接開く（ローカルファイル可）、または
2. 簡易サーバで配信：
   ```bash
   cd desktop-tutorial
   python3 -m http.server 8000
   # ブラウザで http://localhost:8000
   ```

### 操作
- **A**: パンチ（速いが弱い）
- **S**: キック（中威力）
- **D**: 必殺技（必殺ゲージMAXで発動・超威力）
- **F**: ガード（被ダメ大幅減）
- **W**: 突進 / **E**: 投げ（ガード貫通）
- ボタンタップ・スワイプでもOK（スマホ対応）

### 戦闘システム
- **必殺ゲージ**：攻撃ヒット・被弾で溜まる。MAXで必殺技解放
- **ジャストガード**：敵が光って力を溜めた瞬間にガード → 完全無効＋敵気絶＋次の攻撃が確定クリティカル
- **連携技**：コンボ継続中に特定の技順で奥義発動（例：パンチ→パンチ→キック＝喧嘩殺法・三連蹴り）
- **怒りモード**：敵は残りHP35%でキレて攻撃力・速度アップ

### キャラクター
全員リーゼント＋改造学ラン（長ラン＋金ボタン＋ボンタン）のCSS描画スプライト。主人公は黒長ラン＋さらし、ラスボスは白ラン。

## 路線・全23駅

### 名鉄蒲郡線 (蒲郡→吉良吉田) 10駅
蒲郡 → 蒲郡競艇場前 → 三河鹿島 → 形原 → 西浦 → こどもの国 → 東幡豆 → 西幡豆 → 三河鳥羽 → **吉良吉田（中ボス）**

### 名鉄西尾線 (吉良吉田→新安城) 西尾線部分13駅
吉良吉田 → 上横須賀 → 福地 → **西尾（中ボス）** → 西尾口 → 桜町前 → 米津 → 南桜井 → 桜井 → 堀内公園 → 碧海古井 → 南安城 → 北安城 → **新安城（ラスボス）**

## ファイル構成

```
desktop-tutorial/
├── index.html          メイン画面
├── css/style.css       スタイル
├── js/
│   ├── stations.js     23駅 + 不良ボスのデータ
│   ├── scenes.js       駅前ステージ背景データ
│   ├── battle.js       戦闘システム
│   ├── map.js          路線図UI
│   └── game.js         状態管理・画面遷移
└── assets/             （画像差し替え用、将来実装）
    ├── characters/
    └── bg/
```

## 後で画像素材で差し替えたい場合

現状はすべて絵文字＋CSSのプレースホルダ。素材を用意したら：

- キャラ画像 → `assets/characters/{station-id}.png` に置き、battle.jsで`<img>`に差し替え
- 背景画像 → `assets/bg/{station-id}.png` に置き、scenes.jsの該当エントリに `bgImage:` フィールドを追加
